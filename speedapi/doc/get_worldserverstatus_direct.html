Get Server Status

Description

Retrieve game server status.

URI

/ws/game/nfsw/server/status

URI Parameters  | Input Parameters  | Output  | Examples  | Error Codes

URI Parameters
Field Type Required? Example Description

string

string

 Optional

The locale for the localized status message.
shard

string

 Optional

Retrieve information from a specific shard. The default is the configured default shard in server configuration.
