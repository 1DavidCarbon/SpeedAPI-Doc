Get News RSS for Tag

Description

Retrieve an RSS 2.0 feed for the NFSW News filtered by a given Tag

URI

/ws/cmsbridge/news/rss/<locale>/tag<tag>

URI Parameters  | Input Parameters  | Examples

URI Parameters
Field Type Required? Example Description
locale string Required

en_US

The locale to retrieve the news for. e.g. en_US or de_DE.
tag string Required

Competition

The tag to filter the news by. e.g. Competition
[ back to top ]

Input Parameters
Field Type Required? Example Description
firstResult

Integer

 Optional

0

The first result to retrieve (used for pagination).
maxResults

Integer

 Optional

10

The total number of results to retrieve.
[ back to top ]

Examples
Request
uri

/ws/cmsbridge/news/rss/en_US/tag/Competition?firstResult=0&maxResults;=2
Response
status code

200
condition

Success
